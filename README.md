Задания



1.Выберите тему приложения, ориентируясь на простоту реализации.

Генератор случайных фильмов

Мобильное приложение для выбора фильма. Показывает случайные фильмы через API и позволяет сохранять понравившиеся в избранное.

2.Определите основные функции.

Случайный фильм	- Загрузка случайного фильма через API

Добавить в избранное - Сохранение фильма локально

Список избранного - Просмотр сохранённых фильмов

Удалить из избранного - Удаление фильма из списка

3.Нарисуйте эскизы интерфейса (достаточно схематично).

!\[Схема интерфейса](interface.png)

4.Опишите сценарии использования (доступные действия, навигация).

&nbsp;	Сценарий 1: Просмотр случайного фильма

&nbsp;		1. Пользователь запускает приложение

&nbsp;		2. Приложение открывается на главном экране

&nbsp;		3. Пользователь нажимает кнопку "Случайный фильм"

&nbsp;		4. Приложение загружает случайный фильм из API

&nbsp;		5. Отображается информация о фильме

&nbsp;	Сценарий 2: Добавление фильма в избранное

&nbsp;		1. На главном экране пользователь видит информацию о фильме

&nbsp;		2. Пользователь нажимает кнопку "Добавить в избранное" 

&nbsp;		3. Фильм сохраняется в локальное хранилище

&nbsp;		4. Появляется уведомление "Фильм добавлен в избранное"

&nbsp;	Сценарий 3: Просмотр списка избранного

&nbsp;		1. На главном экране пользователь нажимает кнопку "Избранное"

&nbsp;		2. Открывается экран "Избранное"

&nbsp;		3. Отображается список всех сохранённых фильмов

&nbsp;		4. Каждый фильм имеет кнопку "Удалить"

&nbsp;	Сценарий 4: Удаление фильма из избранного

&nbsp;		1. На экране "Избранное" пользователь видит список фильмов

&nbsp;		2. Пользователь нажимает кнопку "Удалить" напротив фильма

&nbsp;		3. Фильм удаляется из списка

&nbsp;		4. Список автоматически обновляется

&nbsp;	Сценарий 5: Возврат на главный экран

&nbsp;		1. Находясь на экране "Избранное"

&nbsp;		2. Пользователь нажимает кнопку "На главную"

&nbsp;		3. Приложение возвращается на главный экран

&nbsp;		4. Отображается генератор случайных фильмов
